# server-express-backend
Простой сервер с api, и базами json-файлами

# Запуск
```
npm run start
```

# Настраиваем переменную окружения в файле `.env` (`файл .env` находится в корне репозитория)

  - так как переменная окружения вызывается так:
```js
const PORT = process.env.PORT || 3001;
```
  то в файле `.env` создаём эту переменную
```
PORT=3002
```

Чтобы она заработала нужно дополнить пакеты зависимостями, установите `dotenv`:
```cmd
npm install dotenv
```

и добавьте в `index.js` строчку для того, чтобы `dotenv` заработало

```js
...
require('dotenv').config();
...
```

если `.env` находится не в корне репозитория, то нужно эту строку дополнить настройками:
```js
...
require('dotenv').config({ path: '/полный/путь/к/файлу/.env' });
...
```

Обращайте внимание, что .env не должно попадать в публичные репозитории, так как содержит пароли, и ссылки, .env попадает в .gitignore
